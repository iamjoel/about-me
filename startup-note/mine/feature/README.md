# 公司特色
## 专业性的体现
* 需求->解决方案
  * 合理的设计。
  * [用户体验](https://github.com/iamjoel/growth-hacking-note/blob/master/pm/ux-checklist.md)。
* 开发进度的可控性。
  * 合理规划的里程碑。
* 程序的稳定性。
  * 代码测试。
    * 单元测试
    * 集成测试

    * [腾讯质量开发平台](http://wetest.qq.com/gaps/)

  * 日志。
* 程序的可维护性。
  * 开发文档。
  * 适当的注释。
  * 代码风格。
    * 风格。
    * 代码格式化工具。
    * 验证工具。ESLint +  npm script precommit 。
