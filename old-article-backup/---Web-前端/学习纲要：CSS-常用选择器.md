> 选择器用来选择要在DOM中哪些元素上使用某些样式。

## 学习目标
1. 知道什么是 CSS 选择器。
1. 知道 CSS 选择器有什么用。
1. 知道如下选择器的写法，对哪些元素生效和浏览器兼容性：通配选择器，元素选择器，id选择器，类选择器，群组选择器，层次选择器，子元素选择器，伪类选择器中的 :hover,:active,:focus,:nth-child，属性选择器。
1. 知道选择器的权重/优先级。
