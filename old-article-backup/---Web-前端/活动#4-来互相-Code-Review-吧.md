Code Review 是一种通过复查代码，提高代码质量的过程。通过 Review 别人的代码，学习别人写的好的地方。别人 Review 自己的代码，别人可以指出自己的不足。

 Code Review 有一堆检查点，我们不用严格地按照那检查点去检查。我们互相 Code Review 主要学习别人写的好的，指出别人写的不好的。

想被 Review 的，把代码地址回复在本文评论区~
